# OCRBench

OCRBench is a benchmark that aggregates multiple in-image text reading tasks (scene text, handwritten text, key information extraction) to measure the OCR quality of an MLLM in a focused way. It provides a single composite score useful for comparing generalist models on this specific ability.

In the wiki it is cited as the reference OCR benchmark alongside [[docvqa]], [[textvqa]] and [[chartqa]].
