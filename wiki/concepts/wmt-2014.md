# WMT 2014

Workshop on Statistical Machine Translation 2014 — provides standard datasets and benchmarks for evaluating machine translation systems. In [[vaswani-2017-attention]] two tasks are used:

- **EN-DE (English → German)**: ~4.5 million sentence pairs; [[byte-pair-encoding]] tokenization with a shared source-target vocabulary of ~37k tokens. Eval on newstest2014 ([[bleu]]: 28.4 for Transformer big) [source: raw/papers/vaswani-2017-attention.pdf §5.1, §6.1].
- **EN-FR (English → French)**: 36M sentences; word-piece tokenization with a 32k-token vocabulary. Eval on newstest2014 (BLEU 41.8 for Transformer big).

Dev set: newstest2013 (used in §6.2 for the ablations).
