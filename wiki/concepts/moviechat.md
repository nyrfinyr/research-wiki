# MovieChat

MovieChat is both a video-LLM system with short/long-term memory and the associated long-video QA benchmark, built on long movie clips where answering requires aggregating information over many minutes. In the wiki it appears in both roles: memory-based competitor for LLoVi and token reduction baseline for AKS.
